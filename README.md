# Presensigps
E-SIMDATSEK adalah aplikasi berbasis web untuk mengelola data dan administrasi sekolah secara terintegrasi, cepat, dan akurat. Fitur meliputi data siswa, guru, jadwal, absensi QR Code, laporan akademik, serta UKS. Sistem ini mendukung efisiensi, transparansi, dan dapat diakses real-time dengan keamanan data yang terjamin.
